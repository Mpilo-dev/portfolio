## 🛠 Portfolio Tech Stack

| Category      | Technologies                                        |
| ------------- | --------------------------------------------------- |
| **Framework** | Gatsby                                              |
| **Styling**   | Styled Components                                   |
| **DevOps**    | AWS (S3, CloudFront, Route 53, Certificate Manager) |

The site is hosted on **AWS**: static assets on **S3**, delivered via **CloudFront**, with **Route 53** for DNS and **Certificate Manager** for HTTPS.

## 🛠 DevOps

- CI/CD (Github Actions)
- AWS(S3, Route 53, Cloudfront, ACM)

## 📦 Getting Started (Local)

```bash
git clone <this-repo>
cd portfolio
npm install
npm run develop
```

3.  **Open the code and start customizing!**

    Your site is now running at http://localhost:8000!
